# Project Idea Generator

A clean, dark‑themed web app that generates random coding project ideas across multiple categories.  
Perfect for developers who want inspiration, students looking for practice projects, or anyone stuck on what to build next.

🔗 **Live Demo!:**  
https://them1ddlem1n.github.io/project-idea-gen/

---

## ✨ Features

- 🎯 Category-based ideas (Python, Web, Games, Tools, Random)
- ⚡ One-click “Generate Idea” and “Surprise” mode
- 🌑 Dark‑only, minimal, modern UI
- ⭐ Favourites system (saved in `localStorage`)
- 📋 Copy idea to clipboard
- 🎨 Smooth animations and responsive layout

---

## 🧰 Tech Stack

- HTML  
- CSS  
- JavaScript (no frameworks)

## 🧠 How It Works
- Ideas are stored in a simple JavaScript object in script.js.

- When you click Generate Idea:

- The app reads the selected category (or picks a random one).

- It selects a random idea from that category.

- It displays the idea with a subtle animation.

- The Surprise button always picks a random category.

- The Favourites system:

- Stores ideas in localStorage

- Lets you add/remove the current idea

- Shows a scrollable list of saved ideas
